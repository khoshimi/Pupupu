# Инструкция по запуску сервера

## Установка зависимостей

Откройте терминал в папке проекта и выполните:

```bash
npm install
```

## Запуск сервера

```bash
npm start
```

Или для разработки с автоматической перезагрузкой:

```bash
npm run dev
```

Сервер будет запущен на `http://localhost:3000`

## Использование

1. Убедитесь, что сервер запущен перед использованием сайта
2. При первом входе пользователь автоматически регистрируется
3. Работы сохраняются в базе данных SQLite (`database.db`)
4. Изображения сохраняются в папке `uploads/`

## API Endpoints

- `POST /api/register` - Регистрация пользователя
- `POST /api/login` - Вход пользователя
- `GET /api/works/user/:userId` - Получить работы пользователя
- `GET /api/works/gallery/:gallery` - Получить работы по галерее (gd, il, wd, zj)
- `POST /api/works` - Добавить работу (с изображением)
- `DELETE /api/works/:workId` - Удалить работу
